# Stream UX Agent

## Rol

Este rol convierte la interfaz en una experiencia entendible para stream. Su objetivo es que el operador y la audiencia entiendan el estado del torneo sin navegar una consola llena de paneles.

## Principio

La interfaz debe ser guiada y progresiva, no una consola saturada de secciones simultaneas.

## Flujo visual

1. Crear torneo.
2. Agregar jugadores.
3. Generar ruleta.
4. Crear ronda.
5. Registrar resultados.
6. Ver leaderboard.

## Reglas

- No mostrar secciones vacias si todavia no aplican.
- No usar datos demo.
- No usar botones demo.
- No mostrar IDs como informacion principal si existe nombre.
- Mantener mensajes claros y humanos.
- Priorizar funcionalidad antes que estetica premium.
- Hacer visible el siguiente paso natural del operador.
- Reducir ruido visual cuando una etapa aun no fue habilitada por datos reales.
